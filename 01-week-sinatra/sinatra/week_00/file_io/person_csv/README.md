#Title: People CSV

###Type:
- CodeAlong/Activity

###Time Required:
- 45 minutes

###Prerequisites:
- Ruby
- Arrays
- Loops
- OOP

# Objective
Learn to read and write from files

# How to lead
Once students can read to and from a file they can refactor their earlier group
lab by reading in the students names from a file and instantiateing student
objects
